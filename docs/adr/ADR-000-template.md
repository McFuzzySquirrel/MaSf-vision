Be specific about:

- What alternatives were considered

---

## How to Use This Template

1. Copy this template for each new ADR
2. Number sequentially (ADR-001, ADR-002, etc.)
3. Use descriptive titles that capture the decision
4. Update status as decision evolves
5. Keep it concise but complete
6. Link related ADRs
7. Date when status changes

## ADR Statuses

- **Proposed**: Under consideration
- **Accepted**: Decision made and being implemented
- **Deprecated**: No longer recommended
- **Superseded**: Replaced by another ADR (link to it)

## Tips for Good ADRs

- Focus on "why" not just "what"
- Document alternatives considered
- Be honest about tradeoffs
- Include relevant context
- Keep it readable
- Update as understanding evolves
